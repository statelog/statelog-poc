# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Statelog, please report it privately.

Please do not disclose security vulnerabilities publicly until they have been reviewed and addressed.

## Contact

For security-related reports, contact:

info@statelog.io

## Supported Versions

This repository is currently maintained as an active proof of concept.

## Scope

This project is published for demonstration and evaluation purposes.

Security reports related to authentication, authorization, token handling, access decisions or rights validation are welcome.
