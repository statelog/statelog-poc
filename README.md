# vol 11

Kliendile presenteeritav kogupakett mitme-tenantilise access decision POC-i jaoks.

## Mis selles paketis on
- `app/` — rakenduse põhikood
- `alembic/` — andmebaasi migratsioonid
- `deploy/` — Docker entrypointid ja deploy abifailid
- `docs/` — lühidokumendid kliendile ja käitusele
- `load/` — latency ja access flow koormustestid
- `tests/` — automatiseeritud testid
- `docker-compose.prod.yml` — production-lähedane käivitusnäide

## Kuidas seda lugeda
1. Alusta failist `docs/EXECUTIVE_SUMMARY.md`
2. Seejärel vaata `docs/SOLUTION_OVERVIEW.md`
3. Tehnilise käivituse jaoks kasuta `README_TECHNICAL.md` ja `docs/DEPLOYMENT_GUIDE.md`

## Kliendivaade
See versioon on puhastatud varasematest vahe-release märkmetest, tühjadest kaustadest ja testijääkidest.
