# Contributing

Thank you for your interest in Statelog.

Statelog is an active proof-of-concept project focused on stateful authorization, digital rights infrastructure and real-time access decisions.

## Reporting Issues

Please use GitHub Issues to:

* Report bugs
* Suggest improvements
* Discuss potential use cases
* Share implementation ideas

## Pull Requests

Pull requests are welcome.

Please:

* Keep changes focused and well documented
* Explain the purpose of the change
* Include tests where appropriate
* Maintain compatibility with existing functionality

## Security Issues

Please do not report security vulnerabilities through public GitHub issues.

Security-related concerns should be reported to:

[info@statelog.ee](mailto:info@statelog.ee)

## Questions

For questions, feedback or collaboration opportunities, open a GitHub Issue or contact:

[info@statelog.ee](mailto:info@statelog.ee)
